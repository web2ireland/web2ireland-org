Any Irish superstars ?

18th of January 2007 by admin

Brad Feld talks about techstars. See Feld Thoughts – TechStars Launches

Similar to ycombinator – the Paul Graham related “startup camp”

In Ireland we have some of the “ingredients”

- EDP programmes
- University activity

but we’re missing out on

- quality advice/mentoring
- angel investing

The exciting thing @ techstars and ycombinator are

1. backed by folks who’ve done it before – “value add”
2. transparent process
3. cluster is together
4. short timeframe
5. escalator model – access to funding if you “graduate”
6. reduces risk for entrepreneur
7. pipeline for angel investors

See here for possible scenarios in Ireland.

We should learn from these programmes….
